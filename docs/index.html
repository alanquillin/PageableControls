Introducing Pageable Controls.

Need reasons to love Pageable Controls? Look no further.

Industry Standard Javascript

Built using JQuery, Pageable Controls implements industry standard javascript that makes it compatible to the latest browsers and html standards.

Responsive

Bootstrap was made to not only look and behave great in the latest desktop browsers (as well as IE7!), but in tablet and smartphone browsers via responsive CSS as well.

Extendable and Customizable

By default the Pageable Controls are built on top of Twitter Bootstrapper to give them a responsive and modern look. However, these controls are not bound to any UI framework and can be customized to to whatever you wish.
